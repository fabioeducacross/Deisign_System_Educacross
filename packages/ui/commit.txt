feat(DataTable): adiciona seleção de linhas e row actions (T9)

FEATURES PRINCIPAIS:
- Row selection com checkboxes (single/multi via TanStack Table)
- Coluna de seleção com checkbox select all no header
- DataTableRowActions: Dropdown com ações contextuais por linha
- Helpers utilitários para gerenciamento de seleção

   - executeBulkAction Executar ação em massa com Promise.all
   - RowAction type Interface para ações customizadas

2. DataTableRowActions.tsx:
   - Dropdown com MoreHorizontal ícone
   - Suporta ícones labels e disabled condicional
   - Separador automático antes de ações destructive
   - Renderiza apenas ações disponíveis não disabled
